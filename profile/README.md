## SpecWorks

The SpecWorks factory uses AI tooling to materialize software parts directly from their specifications. This GitHub organization is intended for hosting repositories that represent software parts.

### Goals

- Attempt to operationalize the theory of am AI powered software factory.
- Provide a place to learn and improve the use AI assisted developer tooling to produce software components that are compliant with specifications and corresponding test suites.
- Produce high quality software components that confirm to publicly available specifications produced by standards organizations and the community.
- Demonstrate the viability of LLM generated software components. Or the opposite.
